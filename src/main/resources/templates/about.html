Welcome to POCOYO
the premier destination for next-generation digital collectibles.
We aren't just an auction house,
we are curators of the metaverse.

This project was designed as a dynamic online auction platform, combining technical robustness with an optimal user experience. Developed with Java Spring Boot and Thymeleaf, it integrates complex features such as dynamic category management, a precise filtering system, and an elegant responsive interface thanks to Tailwind CSS. Beyond the technical aspects, this application demonstrates my ability to structure relational databases and implement concrete business logic. This work is part of my ongoing effort to refine my Fullstack development skills in preparation for entering the corporate world.
